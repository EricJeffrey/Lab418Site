## 进行中的功能
- 常用信息展示
  - 功能细分 - 展示常用信息及时间戳
    - 数据格式:
      ```
      {
          "doc": {
              "title": str,
              "paragraph": str,
              "timestamp": long integer,
          }
      }
      ```
    - 说明: 使用GET请求获取所有信息，展示时根据时间戳由大到小排列文章。修改时直接POST整个，修改后更新时间戳
    - 接口: `/doc GET [doc]`

| ---------------- | ---- | ---- | --------------------------------- |
| 获取所有常用信息 | GET  | 无   | `[{title, paragraph, timestamp}]` |
